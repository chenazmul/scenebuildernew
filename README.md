# Faceless Scene Builder

AI-powered video scene generation tool with support for multiple AI providers.

## Download

Download the latest release from the [Releases](https://github.com/chenazmul/sceanbuildernew/releases) page.

## Features

- AI-powered video scene generation
- Support for multiple AI providers (Gemini, Groq, OpenAI)
- Cinematic scene builder with character consistency
- Voice-over generation in Bengali
- Project management with resume functionality
- Auto-update system

## System Requirements

- Windows 10/11
- Internet connection required
- 4GB RAM minimum (8GB recommended)
